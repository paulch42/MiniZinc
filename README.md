# MiniZinc

This repository contains various artefacts related the the [MiniZinc](https://www.minizinc.org) constraint modelling language. The sub-folders contain:

| Name | Content|
|------|--------|
|[minizinc](minizinc)|A Python MiniZinc parser |
|[flatzinc](flatzinc)|A Python FlatZinc parser|
|[models](models)|Some MiniZinc models|
